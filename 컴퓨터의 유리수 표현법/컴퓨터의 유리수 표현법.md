# 컴퓨터의 유리수 표현법

## 실수
- 2집법 실수
  - 2의 승수 사용
  
  |2^-1|0.5|
  |2^-2|0.25|
  |2^-3|0.125|
  |2^-4|0.0625|
  |2^-5|0.03125|
  |2^-6|0.015625|
  
## 2씩 곱하고 맨앞에 0나오면 0을 1나오면 1을적고 나머지 값이 0인경우 종료
- 예 : 0.1875
0.1875x2 = 0.3750   0
0.3750x2 = 0.7500   0
0.7500x2 = 1.5000   1
0.5000x2 = 1.0000   1

## 정확히 2진수로 표현 못하는 실수가 존재
- 예 : 0.1
